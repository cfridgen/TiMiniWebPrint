# Bundled Web Fonts

These font files are vendored for deterministic text wrapping in the web app.
All fonts were downloaded from the Google Fonts repository.

## Included Fonts

- Inter (`Inter-Variable.ttf`) - OFL-1.1
- Noto Sans (`NotoSans-Variable.ttf`) - OFL-1.1
- Noto Serif (`NotoSerif-Variable.ttf`) - OFL-1.1
- Source Serif 4 (`SourceSerif4-Variable.ttf`) - OFL-1.1
- Roboto Mono (`RobotoMono-Variable.ttf`) - OFL-1.1
- IBM Plex Mono (`IBMPlexMono-Regular.ttf`) - OFL-1.1
- Courier Prime (`CourierPrime-Regular.ttf`) - OFL-1.1
- Cutive Mono (`CutiveMono-Regular.ttf`) - OFL-1.1
