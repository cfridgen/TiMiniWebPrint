# ThermoFlow Print Brand Guidelines

This document defines how to use the ThermoFlow Print visual identity consistently.

## Official product name

- Primary name: ThermoFlow Print
- Do not use mixed variants in user-facing text.
- Keep technical/internal names unchanged where required (`timiniprint`, image tags, module paths).

## Logo files

- `thermoflow_wordmark_dark.png`
  - Use on dark backgrounds.
  - Preferred for README hero and dark website sections.

- `thermoflow_wordmark_light.png`
  - Use on light backgrounds.
  - Preferred for docs pages with white/light backgrounds.

- `thermoflow_appicon.png`
  - Use as square icon for app/profile/avatar contexts.
  - Works well in 512x512 and 1024x1024 exports.

- `EMX_040256.jpg`
  - Product hero banner image.
  - Use where a full-width visual is needed.

## Background rules

- Dark background: use `thermoflow_wordmark_dark.png`.
- Light background: use `thermoflow_wordmark_light.png`.
- Avoid placing logos over busy backgrounds unless a solid panel is used.

## Spacing and sizing

- Keep clear space around logo at least equal to the printer-button diameter in the mark.
- Minimum recommended width:
  - Wordmark logos: 280 px
  - App icon: 64 px

## Color direction

Use these reference tones for consistency:

- Primary navy: `#121B34`
- Accent blue: `#4A7EE6`
- Accent teal: `#2DB2A2`
- Light neutral: `#F5F8FF`

## Copy style (user-facing)

- Keep tone practical and simple.
- Focus on user outcomes (scan, connect, print).
- Avoid developer-heavy language in top sections.
